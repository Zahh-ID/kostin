# ModerationActionController

**Lokasi Kode**: `App\Http\Controllers\Api\V1\Admin`

## Penjelasan Singkat
Bagian ini berfungsi untuk mengatur segala hal yang berkaitan dengan **ModerationAction**.

## Daftar Fungsi
- **`approve`**: Menyetujui permintaan atau data.
- **`reject`**: Menolak permintaan atau data.
